Classification Of Internet

❇ 2G

2G = (2nd Generation) দ্বিতীয় প্রজন্মের ইন্টারনেট সেবা। যা GSM (Global System for Mobile) প্রযুক্তির মাধ্যমে পরিচালিত। GSM প্রতি সেকেন্ডে 10-40kb ডাটা প্রদান করে।


2.5G বা GPRS (General Packet Radio Service)

2.5G = GPRS (General Packet Radio Service) GPRS প্রতি সেকেন্ডে 56-114kb ডাটা প্রদান করতে সক্ষম।


2.75G বা EDGE (Enhanced Data Rate for GSM Evolution)

2.75G = EDGE (Enhanced Data Rate for GSM Evolution) উন্নত ডাটা রেট GSM বিবর্তনের জন্য। EDGE প্রতি সেকেন্ডে 400Kb পর্যন্ত ডাটা প্রদান করতে সক্ষম।


❇ 3G

3G = (Third Generation) তৃতীয় প্রজন্মের ইন্টারনেট সেবা। যা WCDMA-(UMTS) প্রযুক্তির মাধ্যমে পরিচালনা করা হয়। WCDMA= (Wideband Code Division Multiple Access) যার গতি EDGE এর চেয়ে কিছুটা বেশী।


3.5G বা HSDPA (High Speed Downlink Packet Access)

3.5G = HSDPA (High Speed Downlink Packet Access) এর মাধ্যমে প্রতি সেকেন্ডে 7.2Mb ডাটা পাওয়া সম্ভব। HSUPA(High Speed Uplink Packet Access) HSUPA প্রতি সেকেন্ডে 5.8 Mb আপলোড স্পীড পাওয়া যায়।


3.75G বা HSPA (High Speed Packet Access)

3.75G = HSPA (High Speed Packet Access) হাই স্পিড প্যাকেট এক্সেস HSPA প্রযুক্তিতে প্রতি সেকেন্ডে 21 থেকে 28Mb স্পিড পাওয়া সম্ভব।


3.8G, 3.85G, 3.9G (Pre-4G)

3.8G, 3.85G, 3.9G (Pre-4G) = HSPA+ (Evolved High Speed Packet Access) প্রসূত হাই স্পিড প্যাকেট এক্সেস। HSPA+ প্রযুক্তিতে প্রতি সেকেন্ডে 168mb পর্যন্ত স্পিড পাওয়া সম্ভব।


❇ 4G

4G = (Fourth Generation) চতুর্থ প্রজন্ম LTE (Long Term Evolution) দীর্ঘমেয়াদী বিবর্তন। LTE প্রযুক্তিতে প্রতি সেকেন্ডে 299.6Mb পর্যন্ত গতি পাওয়া সম্ভব।



❇ উল্লেখ্য,

আপনি যদি 3G নেটওয়ার্ক এলাকায় অবস্থান করেন, তবে আপনার মোবাইলে 3G লেখা দেখাবে, আপনি যদি 3.5G বা 3.75G নেটওয়ার্ক এলাকায় অবস্থান করেন, তবে আপনার মোবাইলে H লেখা দেখাবে, আর আপনি যদি 3.8G বা 3.85G বা 3.9G নেটওয়ার্ক এলাকায় অবস্থান করেন, তবে আপনার মোবাইলে H+ লেখা দেখাবে।
